PROJECT PLANS
=================
This section describes all project plan related procedures at SADiLaR.

A project plan procedure is triggered when:
..*An open call has been approved by evaluatators
..*A node project has been approved by the Scientific Advisory Committee
..*A hub project has been approved

##High level process:##
Based on the project proposal document / business plan an annexure is drafted using the SADiLaR Annexure Template - Contracts.SADiLaR.Annexure.ProjectPlan.1.0.ZJS - available at NextCloud/Projects/Templates.
